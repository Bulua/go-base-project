# 02-用户管理模块

## 1. 模块职责

负责后台账号生命周期管理，包括创建用户、编辑用户、冻结用户、重置密码、删除用户、分配角色和查询用户列表。

## 2. 涉及数据表

```text
gbp_users
gbp_roles
gbp_user_roles
gbp_operation_audit_logs
```

## 3. 核心接口

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | /api/v1/users | 用户分页列表 |
| GET | /api/v1/users/:id | 用户详情 |
| POST | /api/v1/users | 创建用户 |
| PUT | /api/v1/users/:id | 修改用户 |
| DELETE | /api/v1/users/:id | 删除用户 |
| PUT | /api/v1/users/:id/status | 冻结/解冻用户 |
| PUT | /api/v1/users/:id/password | 重置密码 |
| PUT | /api/v1/users/:id/roles | 分配角色 |

## 4. 关键流程

```text
创建用户并校验账号 -> 校验密码强度 -> 生成 UUID -> 哈希密码 -> 写用户表 -> 写角色关系 -> 记录操作日志
```

## 5. 开发注意事项

1. 用户删除使用软删除。
2. admin 内置用户不允许删除。
3. 重置密码后建议强制用户下次登录修改密码。
4. 修改角色时应使用事务。
