# 08-字典管理模块

## 1. 模块职责

负责维护系统中通用枚举、状态、下拉选项和级联选项。

## 2. 涉及数据表

```text
gbp_dictionaries
gbp_dictionary_items
```

## 3. 核心接口

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | /api/v1/dictionaries | 字典列表 |
| POST | /api/v1/dictionaries | 创建字典 |
| PUT | /api/v1/dictionaries/:id | 修改字典 |
| DELETE | /api/v1/dictionaries/:id | 删除字典 |
| GET | /api/v1/dictionaries/:dictCode/items | 根据编码查询字典项 |
| POST | /api/v1/dictionaries/:id/items | 创建字典项 |
| PUT | /api/v1/dictionary-items/:id | 修改字典项 |
| DELETE | /api/v1/dictionary-items/:id | 删除字典项 |

## 4. 关键流程

```text
维护字典 -> 维护字典项 -> 前端 useDict(dictCode) 拉取 -> 页面下拉/状态渲染
```

## 5. 开发注意事项

1. dict_code 全局唯一。
2. 字典项支持树形结构。
3. 字典适合后端和前端缓存，但需要刷新机制。
