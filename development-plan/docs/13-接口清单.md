# 13-接口清单

## 认证鉴权

| 方法 | 路径 | 说明 | 鉴权 |
|---|---|---|---|
| POST | /api/v1/auth/login | 登录 | 否 |
| POST | /api/v1/auth/logout | 退出 | 是 |
| POST | /api/v1/auth/refresh | 刷新 Token | 是/可选 |
| GET | /api/v1/auth/profile | 当前用户信息 | 是 |
| GET | /api/v1/auth/routes | 当前用户动态路由 | 是 |
| GET | /api/v1/auth/actions | 当前用户按钮权限 | 是 |
